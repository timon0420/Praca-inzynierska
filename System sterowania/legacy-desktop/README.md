# Archiwalna aplikacja desktopowa

Ten katalog zawiera historyczny interfejs PyQt, obsługę lokalnej kamery i klienta
WebSocket `/ws/python`. Kod nie jest częścią aktywnego systemu, nie jest budowany,
testowany ani wdrażany.

Aktywny system składa się z aplikacji webowej, serwera Go, workera Python/MediaPipe
oraz Unity. Kalkulator pozycji dłoni znajduje się w `backend/hand_position.py`.
